# secure-passwords

Secure password generator with a Tkinter GUI and multiple generation strategies.

## Requirements

- Python 3.14+
- Pipenv

## Setup

```bash
pipenv install --dev
```

## Common commands

```bash
make all
make test
make coverage
make build
make clean
```

## Run the application

```bash
PYTHONPATH=src pipenv run python -m secure_passwords
```

## Project structure

- src/secure_passwords: package source code
- tests: unit tests
- Makefile: development, quality, coverage and build tasks
